Код завершения программы

Функция выполняет моделирование системы управления с ПИД-регулятором. Последовательность работы:

  • Ввод параметров модели и регулятора
  • Инициализация переменных состояния
  • Расчет коэффициентов ПИД-регулятора
  • Цикл моделирования на 50 шагов
  • Вывод промежуточных результатов
Заметки
Используется нелинейная модель объекта: y(k) = a*y(k-1) - b*y(k-2)^2 + c*u(k-1) + d*sin(u(k-2))

Параметры нелинейной модели объекта a, b, c, d

Начальные значения температуры y[0] и y[1]

Параметры ПИД-регулятора: K - коэффициент усиления, T - постоянная времени, Td - постоянная времени дифференцирования

Заданное значение температуры (уставка)

Массив температур: y[0], y[1], y[2] - история температур (y(k), y(k-1), y(k-2))

Массив управлений: u[0], u[1] - история управляющих воздействий (u(k), u(k-1))

Предыдущее управляющее воздействие u(k-1)

Предыдущая ошибка регулирования e(k-1)

Предпредыдущая ошибка регулирования e(k-2)

Шаг квантования (время дискретизации)

Коэффициенты ПИД-регулятора в дискретной форме

Основной цикл моделирования системы управления на 50 шагов

Текущая ошибка регулирования e(k) = setpoint - y(k-1)

Управляющее воздействие u(k) = u(k-1) + q0*e(k) + q1*e(k-1) + q2*e(k-2)

Новая температура по нелинейной модели y(k) = a*y(k-1) - b*y(k-2)^2 + c*u(k-1) + d*sin(u(k-2))

Вывод результатов каждые 10 шагов для мониторинга
